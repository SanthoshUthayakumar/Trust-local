Trust-Local
Location-Based Worker Booking & Trust Platform

Project Overview
Trust-Local is a full-stack web application that connects users with verified local workers. The platform focuses on trust, safety, rating systems, and location-based hiring.

Key Features
•	User & Worker Registration with document verification
•	Admin approval system with initial 40 Honest Score
•	Location-based filtering (City, Area, Pincode)
•	Workers can post skills with hourly rate
•	Users can book workers by selecting number of hours
•	Automatic total payment calculation
•	Request → Accept → Job workflow
•	Two-way rating system (User & Worker)
•	Honest score increases or decreases based on ratings
•	Worker availability scheduling system

Technology Stack
Frontend: React.js, Axios, CSS
Backend: Node.js, Express.js, Multer
Database: MySQL

Future Enhancements

Project Objective
The objective of Trust-Local is to build a trusted local employment ecosystem where verified workers can find jobs and users can hire safely with transparent ratings and honor scores.
